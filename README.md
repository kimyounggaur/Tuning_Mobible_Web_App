# TuneString

설치 없이 브라우저에서 실행되는 현악기 튜너 PWA입니다. 기타, 베이스, 우쿨렐레, 바이올린, 비올라, 첼로, 콘트라베이스, 만돌린, 밴조와 크로매틱 모드를 지원합니다.

## 실행

```bash
npm install
npm run dev
```

빌드 확인:

```bash
npm run build
npm run preview
```

마이크 입력은 HTTPS 또는 localhost에서만 동작합니다.

## 상수 튜닝 가이드

| 상수 | 기본값 | 역할 | 올릴 때 | 내릴 때 |
|---|---:|---|---|---|
| `CLARITY_MIN` | `0.90` | NSDF 피크 명료도 컷 | 말소리·생활소음 반응 감소, 약한 악기음 누락 가능 | 약한 입력 감지 증가, 소음 오작동 가능 |
| `RMS_MIN` | `0.008` | 무음 컷 기준 | 주변 소음 반응 감소, 작은 소리 누락 가능 | 작은 소리 감지 증가, 무음 상태 흔들림 가능 |
| `RECENT_FREQ_SIZE` | `5` | 최근 주파수 중앙값 창 | 바늘 안정성 증가, 반응 지연 증가 | 반응 빨라짐, 지터 증가 |
| `NOTE_LOCK_COUNT` | `3` | 크로매틱 노트 변경 확정 횟수 | 노트 깜빡임 감소, 전환 지연 증가 | 빠른 전환, 경계 음에서 깜빡임 증가 |
| `STRING_LOCK_COUNT` | `3` | 자동 현 변경 확정 횟수 | 현 오인식 감소, 현 전환 지연 증가 | 빠른 현 감지, 인접 현 오인식 증가 |
| `IN_TUNE_CENTS` | `5` | 인튠 진입 허용 오차 | 성공 판정 쉬움, 정밀도 하락 | 정밀해짐, 초보자 난도 상승 |
| `IN_TUNE_HOLD_MS` | `500` | 인튠 유지 필요 시간 | 우연한 순간 판정 감소, 보상 지연 | 보상 빨라짐, 경계 흔들림에서 오판 가능 |
